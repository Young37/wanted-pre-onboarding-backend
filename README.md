# wanted-pre-onboarding-backend
10월 백엔드 인턴십 - 사전 과제

# 프로젝트 개요
이 프로젝트는 기업의 채용을 위한 웹 서비스를 제공하는 백엔드 API 서버를 구현하는 것을 목표로 합니다.<br><br>
사용자는 채용공고를 등록, 수정, 삭제하고 채용공고 목록을 확인할 수 있으며, <br>
채용공고 검색 및 채용 상세 정보를 조회할 수 있습니다.<br>
사용자는 또한 채용공고에 지원할 수 있습니다.
